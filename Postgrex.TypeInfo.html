Postgrex.TypeInfo

  • Types

The information about a type that is provided to the custom encoder/decoder functions.

  • oid - The type's oid;
  • sender - The name of the "sender" function (the function postgres uses to convert the type to binary format) without the "send" postfix. Useful to identify types that share a common binary format by a common name;
  • type - The type name;
  • array_elem - If the type is an array, the array elements' oid;
  • comp_elems - If the type is a composite type (record), the tuple elements' oid;
